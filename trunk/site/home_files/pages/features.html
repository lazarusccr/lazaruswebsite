Features

Lazarus is packed with features that might satisfy you:

Freedom

Simple, compatible performance

  • Everything necessary to get started is installed by default
  • Your program stays compatible to major (and minor) platforms (Windows, MacOS, Linux, BSD & more)
  • Make modifications, rename components and Lazarus will update the code automatically
  • Share the same code with anyone. Static linking would let them compile easily, no matter what platform they use
  • GPL license allows you to make commercial apps or do whatever you feel doing!

Peace

Easy, uniform
experience

  • Easy Drag & Drop Form designer
  • Components have the same behavior on different platforms
  • Develop and code your program the same way between platforms
  • Uniform looking forms throughout platforms
  • No fuss for users
    • No platform dependencies
    • No frameworks or virtual machines (unless you want to use them)
    • Makes it simple to run your softwares, just like a native app

Creativity

Modern, customizable productivity

  • Supports modern technologies, such as, multithreading, UTF-8 etc.
  • Elaborate compiler options and numerous component properties lets your customize up to the last nuts and bolts
  • Be productive with over 200 components for dates, colors, inputs, databases, highlighting, charting, outputs, layouts etc.
  • Further features with Lazarus Packages (LPKs)

Cross Platform

Lazarus runs on Windows, MacOS, Linux and many other platforms. Even on Raspberry Pi! The programs you create also runs on these platforms. Enjoy the same experience on your favorite OS. See Installing Lazarus for OS-specific instructions for installation.

Powerful, Fast

Lazarus compiles, loads and performs faster than its commercial alternative. Its compiler FPC is being constantly developed to improve performance. Lazarus is a capable IDE for handling large projects. Lazarus is made by Lazarus itself. So you can guess.

Drag and Drop Form Designer

Lazarus has a graphical form designer with guidelines for aligning with adjucent components. There are numerous components for almost every software imaginable, already ready to be used. Further components can be added to the IDE by Lazarus Package Files (LPKs).

Easily deployable

If you are tired of installing frameworks, so are we. Lazarus links the libraries statically. Which means your executable file does not need any dll files or frameworks to run, unless you want to use them.

Cross-platform but uniform experience

Lazarus form designer uses LCL (Lazarus Component Library) which is especially designed for cross platform performance in mind. Give your users an uniform experience in all platforms.

Modern, Object oriented language

Lazarus uses Free Pascal as its language which is an Object Pascal dialect. It is an Object oriented High level language which is very appropriate for cross-platform compiling.

Human readable coding style

Programing Languages can be hard to read. Free Pascal is a language which heavily uses dictionary words, such as begin, end, uses, interface, implementation, read, write etc. which makes codes easily readable and understandable.

Light on space

Executables made with Lazarus have no dependencies which frees you from installing huge frameworks and virtual machines. That lets you store more of your stuff rather than just struggle for the software.

Easily Extendable

Hundreds of developers are developing and maintaining their Lazarus Libraries and Lazarus Packages (LPKs). You can find almost any kind of library that you may need.

Open source, fear free

Lazarus is open source and LGPL/GPL, which is fearless. So you can create non-commercial and commercial applications with it.

Frameworks

Various Frameworks are available which saves your time on certain codebase.
